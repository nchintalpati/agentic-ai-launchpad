# Agentic AI Launchpad

A hands-on learning repository for building Agentic AI systems using modern tools and frameworks.

## Modules

| Module | Description |
|--------|-------------|
| [1-env-setup](./1-env-setup/) | Environment setup and initial configuration |

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/nchintalpati/agentic-ai-launchpad.git
   cd agentic-ai-launchpad
   ```

2. Start with the [Environment Setup](./1-env-setup/) module

## Tech Stack

- **Python 3.11** - Core language
- **LangChain** - LLM orchestration framework
- **Google Gemini** - AI model provider
- **GitHub Actions** - CI/CD automation
